# 📱 Mis Tareas - React Native App

Aplicacion para gestionar tareas con calendario

## 🚀 Características

- Gestión de tareas con categorías 
- Calendario interactivo
- Diseño responsive
- Guardado de datos
- Marcado de tareas completadas

## 🛠️ Tecnologías

- React Native
- Expo
- TypeScript
- AsyncStorage

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/helenaSarrion/MisTareas

# Instalar dependencias
npm install

# Ejecutar
npx expo start
